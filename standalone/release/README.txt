# OC Proxy Downloader - Standalone 
 
실행: oc-proxy-downloader.exe 
웹 인터페이스: http://localhost:8759 
 
다운로드 폴더: 사용자 다운로드 폴더 (C:\Users\사용자명\Downloads) 
config/ - 설정 저장 폴더 
 
주의: 첫 실행 시 Windows Defender에서 차단할 수 있습니다. 
     "추가 정보" > "실행" 을 클릭하여 실행하세요. 
 
포트: 8759 (웹 버전과 충돌 방지) 
버전: Standalone EXE v1.0
